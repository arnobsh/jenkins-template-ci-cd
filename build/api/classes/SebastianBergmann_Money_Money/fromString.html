SebastianBergmann\Money\Money::fromString

Creates a Money object from a string such as "12.34"

This method is designed to take into account the errors that can arise from manipulating floating point numbers. If the number of decimals in the string is higher than the currency's number of fractional digits then the value will be rounded to the currency's number of fractional digits.
